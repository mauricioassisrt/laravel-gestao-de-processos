# Projeto Base
Detalhes do app

## Technical Characteristics:
- PHP 7.1
- Framework Laravel 5.8
- Datatables
- Javascript, JQuery
- CSS 3.0, Bootstrap 4
- Composer
- Mysql 5.7
- SMTP Server
- SSL TSL 1.2

## Install (Docker)

bin/up

## Install (No VM)

$ composer install

$ cp .env.example .env

$ php artisan key:generate

$ php artisan migrate:refresh --seed

$ php artisan serve
